# Moonlight Affirmations

A small React app with affirmations, favorites, TTS playback, import/export, and simple browser reminders.
Styled with a night/midnight theme (moonlight colors) and soft glow UI elements.

## Quick start
1. npm install
2. npm run dev
3. open http://localhost:5173

## Assets
- logo SVGs in src/assets/
- app icons in src/assets/icons/
